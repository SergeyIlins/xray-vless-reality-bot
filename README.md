# 🚀 Xray VLESS+REALITY + Telegram-бот

Полнофункциональный VPN-сервер на базе **Xray (VLESS + XTLS-Vision + REALITY)** с удобным управлением через Telegram-бота.  
Идеальное решение для обхода блокировки и организации приватного доступа в интернет.

## ✨ Возможности

- 🔧 **Автоматическая установка** – один скрипт устанавливает Xray, Python-окружение, все зависимости и сервисы.
- 🤖 **Управление через Telegram** – добавление, удаление, просмотр списка клиентов и статистики прямо из мессенджера.
- ⏱️ **Гибкие сроки действия** – выдавайте доступ на 24 часа, месяц, год или навсегда.
- 🧹 **Автоочистка** – просроченные клиенты удаляются автоматически (systemd-таймер).
- 📲 **Готовые конфигурации** – после добавления клиента бот присылает:
  - 🔗 прямую ссылку `vless://` для импорта
  - 📷 QR-код
  - 📄 файл `.conf` для ручной настройки
- 🛡️ **Маскировка под HTTPS** – технология REALITY заимствует сертификат реального сайта, делая трафик неотличимым от обычного веб-сёрфинга.
- 📊 **Мониторинг трафика** – бот показывает суммарный объём переданных данных по каждому пользователю.
- 🌍 **Обход блокировок** – связка VLESS + XTLS-Vision + REALITY эффективно противостоит системам блокировки.

## 📦 Быстрый старт

```bash
git clone https://github.com/SergeyIlins/xray-vless-reality-bot
cd xray-vless-reality-bot
sudo bash install.sh
Скрипт задаст несколько вопросов:

Токен Telegram-бота (получить у @BotFather)
Ваш Telegram ID (узнать можно через @userinfobot)
IP‑адрес сервера (определится автоматически, нажать Enter)

После завершения установки бот сразу готов к работе. Откройте чат с ботом и нажмите /start.

📱 Рекомендуемые клиенты
Платформа	Приложения
Android	v2rayNG, Hiddify
iOS	Shadowrocket, FoXray
Windows	v2rayN, Nekoray
macOS	Nekoray, V2RayX
Linux	Nekoray

🛠️ Ручная настройка (если нужно)
После установки конфигурационные файлы находятся в:
Конфиг Xray: /usr/local/etc/xray/config.json
Файл с настройками бота: /opt/xray-bot/config.py
База клиентов: /opt/xray-bot/clients.json

Для перезапуска сервисов:

bash
systemctl restart xray        # перезапуск Xray
systemctl restart xray-bot    # перезапуск Telegram-бота

## 🩺 Диагностика проблем

Если после установки VPN не подключается, воспользуйтесь встроенным диагностическим скриптом:

```bash
sudo bash xray_full_test.sh

📋 Системные требования
ОС: Debian 12 / Ubuntu 22.04+ (рекомендуется)

Порты: открытый TCP 443 (основной)
Память: от 512 МБ ОЗУ

Домен не требуется – REALITY использует подставной SNI

🔒 Безопасность
🔐 Приватные ключи генерируются локально на сервере и никогда не передаются в репозиторий.

🚫 Доступ к боту ограничен списком Telegram ID (задаётся в config.py).
